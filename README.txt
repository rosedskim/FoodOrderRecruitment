--설치--
pip install flask
pip install pyodbc
pip install pandas
pip install sklearn
pip install scipy

python 3에서 작동합니다.
원활한 실행을 위해서는 DB 스키마 구조를 설정하셔야합니다.

--실행--
python flask를 실행시킵니다.
실행시키면 나오는 주소에 접속합니다.
아이디가 있으면 로그인, 아이디가 없으면 회원가입합니다.
아직 대학교 범위를 크게 넓히지 않았으므로, 대학교란에는 '경희대학교'만 입력 가능합니다.
경희대학교가 아닌 다른 대학교 입력시 회원가입이 진행되지 않습니다.
로그인합니다.
로그인에 성공하면 메인화면으로 진행됩니다.
메인화면에서는 추천메뉴, 누계, 방목록, 방생성, 로그아웃을 진행할 수 있습니다.

-방생성
한식, 족발보쌈, 중식, 야식, 치킨, 피자, 분식, 기타 등의 목록으로 음식점이 나뉘어 있습니다.
원하는 카테고리를 선택한 뒤 음식점 이름을 클릭하면 메뉴를 볼 수 있으며, 음식점을 선택하거나 메뉴창을 닫을 수 있습니다.
음식점 선택시 화면 좌상단의 음식점란에 이름이 자동으로 추가됩니다.
직접 음식점 이름을 추가할 수 있으나, 정확히 이름이 입력되지 않으면 방 생성이 진행되지 않습니다.
방 제목, 비밀번호, 인원 수, 음식점을 입력한 뒤 방을 생성합니다.
다시 메인화면으로 이동하여 방을 클릭하면, 방의 비밀번호를 입력할 수 있습니다.
방의 비밀번호를 입력하면 방에 입장할 수 있습니다.

방에 입장하면 모든 사용자는 방 정보와 방 생성시 선택한 음식점 정보, 주문준비, 메뉴선택, 방나가기를 선택할 수 있습니다. 
방장은 추가적으로 최종주문결정과 결제자를 결정할 수 있습니다.
사용자들이 메뉴선택을 누르면, 해당 음식점의 메뉴목록이 뜨며, checkbox로 메뉴를 선택할 수 있습니다.
메뉴를 선택하고 모든 사용자가 주문준비를 마치면 방장은 결제자를 결정합니다.
방장이 결제자 결정까지 모두 마치게 되면, 방장이 최종주문결정을 할 수 있습니다.
